# Advance-Peripheral-Bus

## 📌 Overview

The **Advanced Peripheral Bus (APB)** is a part of the **ARM Advanced Microcontroller Bus Architecture (AMBA)** family of on-chip communication protocols. It is specifically designed for connecting low-bandwidth, low-power peripherals such as timers, UARTs, GPIOs, and other simple control registers within a System-on-Chip (SoC).
